# Privacy

GitHub PR Lifecycle Filter does not collect, transmit, sell, or share personal data.

The extension reads the current GitHub pull-request search query and GitHub's rendered
Open/Closed count so it can display and apply the selected lifecycle view. It stores each
customized menu layout locally on the device under that repository's `owner/name` identifier.
These preferences are shared by the browser between normal and private windows, but are not
synced to a browser account or transmitted anywhere by the extension.

The extension has no analytics, remote service, backend, authentication flow, or GitHub API
integration. It declares `none` for Firefox data collection and transmission.
